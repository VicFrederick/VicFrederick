How to Make a Free Website

In this article I will discribe how to make a website for free just like this one. I will walk through basic HTML, CSS, Github Pages, and creating a custom domain.

HTML

Hypertext Markup Language(HTML) allows you to write the content of your website. The basic example from w3schools gives the framework of an HTML page. Open the example and save the text as a ".html" page. The page will open in a brouser by default when double clicked on. To edit the page, open from a text editor or right click the file and scroll down to "Open with" to find the text editor of your choice.

CSS

Cascading Style Sheets(CSS) allows you to format the content of your website Again, w3schools is going to be your best resoruce. The two different types of CSS are going to be in-line and external.

-In-Line Style

When you write the opening tag for a piece of content, a "style = cssFormatting" can be instered before the closing carrot. This allows you to style that specific content with no effect on the rest of the document

-External Style Sheet

Alternatively within the head of your HTML document, you can include a styles sheet. Adding the style sheet is as simple as writing the following link <link href="styleSheetName.css" rel="stylesheet">

The format of the external style sheet follows the in-line css, but for all HTML and CSS progress going forward, I recommend using w3school librally.

Housting on GitHub Pages

Now that you have writen a website, it is time to make it availible to the world. Github allows you to host websites from respositories using the pages feature.

  1. Start a new pulic respository
  2. Add wesite documents to the repository renaming the first page "index.html"
  3. Go to "settings" and scroll down to "GitHub Pages"
  4. Under the source drop down select "master branch"
  5. Click live link to visit website

Creating a Custom Domain

Free Domains can be found on freenom.com Finding your domain is as simple as typing the name of your desired URL into the search bar. Next, select an extention. The free extentions will be availivble to your for 12 months, but you can always renew down the line. You need the IP address of your host to use this domain. Go to DNS Configuration and follow the instructions from Freenom here. To set up the networking from Freenom to GitHub you need to add all four IPs to the A Dom in Freenom. Lastly, declare your domain by adding a file named "CNAME" to your repository with the first line of that file being your domain.

After following these steps you should end up with a website as live as this one. Happy trouble shooting!
